## Como rodar o repositório
### Clone o repositório
```sh
git clone https://github.com/ryan-wakugawa/vue-tutorial.git
```
### Vá até o diretório
```sh
cd vue-tutorial
```
### Instale as dependencias
```sh
npm i
```
### Rode o backend
```sh
npm run server
```
### Em um outro terminal, rode a aplicação
```sh
npm run dev
```
